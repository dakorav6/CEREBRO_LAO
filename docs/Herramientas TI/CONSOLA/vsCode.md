

# Atajos de teclado

- Abrir el Terminal: ` Ctrl + j ` 



# Lista de Comandos VS CODE
- cd <nombre-carpeta>: Entra en una carpeta.
- cd ..: Retrocede un nivel (vuelve a la carpeta anterior).
- clear: Limpia la pantalla de la terminal para que no se vea desordenada.

- mkdir <nombre>: Crea una carpeta nueva.
- touch <nombre-archivo.ts>: Crea un archivo
