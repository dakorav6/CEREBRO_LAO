# POWER BI

- es una herramientas de visualizacion de datos creado por microsoft
- creacion de dashboard interactivos con bussnis inteligen 
	- Un **dashboard** (o tablero) es una herramienta visual que muestra los datos más importantes de un negocio en una sola pantalla. Su objetivo es que puedas entender qué está pasando de un solo vistazo.
	- un dashboard en Power BI te permite interactuar:Si haces clic en un país, todos los gráficos cambian para mostrarte solo los datos de ese país.Puedes filtrar por fechas (por ejemplo, ver solo las ventas de la última semana).


GLOSARIO 
- UTILIDAD: BENEFICIO (INGRESOS - GASTOS)
- MARGEN: % PROPORCION RESPECTO A INGRESOS (UTILIDAD / INGRESOS )
